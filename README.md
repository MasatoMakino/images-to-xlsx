# images-to-xlsx

.xlsx image list generator.

## Getting Started

### Install

```bash
npm install https://github.com/MasatoMakino/images-to-xlsx.git --save-dev
```

### Run

```bash
npm run images-to-xlsx -i ./images
```
load images in directory ``./images`` and output ``./images.xlsx``

### Options

##### -i --input

image directory.

##### -o --output [optional]

output file path.

default : ``./images.xlsx``

##### -h

show help.
